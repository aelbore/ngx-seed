# ngx-seed
Angular Application Seed

## Getting Started 

* Clone ngx-seed repository
  ```
  git clone https://github.com/aelbore/ngx-seed.git
  ```
* Install dependencies
  ```
   npm install
  ```
* Start the Application
  ```
  npm start
  ```
  - this will build, watch the file changes, and live reload
